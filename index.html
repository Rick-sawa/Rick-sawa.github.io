Master's Research: Inverse Problems in Medical CT

                   

                        Next-generation medical imaging, like Photon-Counting CT (PCCT), is becoming important. My research aims to solve one of the major problems of PCCT, called **"Pulse Pile-up"**, by using an inverse problem approach. Pulse pile-up happens when multiple photons are counted as one large-energy photon. I create a mathematical model for the physical process inside the semiconductor detector. Then, I use statistical techniques like Bayesian inference and sparse estimation to accurately estimate the original signal, which helps to improve the quality of the CT images.

Bachelor's Research: Development of Image-based Strain Measurement Method

                   

                        I developed a non-contact method to precisely measure very small strains (deformations) on the surface of objects, which are too small to see with eyes. The key idea of this method is combining a special mechanism, called a **Compliant Mechanism** (which physically magnifies the strain), with image processing technology. Through experiments, I showed that this approach is effective for improving measurement accuracy.

Image Recognition Sorting Robot

                   

I developed a robot that automatically sorts specific objects by combining machine learning for image recognition and classical control for line tracing. The robot identifies a dog or cat from a camera image, then uses an arm to grasp the object, and drives autonomously along the line to the correct location for precise sorting.

               
           
           
               
                                   
               
                   

Two-Wheel Inverted Pendulum Robot

                   

I designed and implemented a **PID control** system that maintains stable inversion using the angular velocity from a gyro sensor and travel information from a motor encoder. It calculates the optimal motor output in real-time based on the sensor values to prevent falling, achieving autonomous self-balancing.

               
           
           
               
                                   
               
                   

Gyroscope Two-Wheel Robot

                   

This two-wheel robot was created when I represented Yamanashi Prefecture in the Science Koshien National Competition. Unlike a typical two-wheeler, its key feature is that it uses the **gyro effect** of a high-speed rotating disk to maintain posture and achieve autonomous driving. The unique mechanical design and control led to a high evaluation in the competition.
